## 描述:
*  一个windows 在控制台下的小游戏


## 按键:
*    上-下-左-右 控制方向/菜单选择
*   <ESC>       退出游戏
*   <R>         重新开始
*   <S>         游戏保存
*   <L>         游戏读取
*   <SPACE>     暂停
*   <M>         声音开关切换


## 胜利条件:
*   当蛇的长度达到[ 30 ]时即获得游戏胜利

## *Tips*:
* 按键如果和蛇头的方向一样,可加速
